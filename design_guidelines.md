# 记忆典当行 - 设计指南

## 品牌定位
- 应用类型：赛博朋克风叙事冒险游戏
- 设计风格：黑色电影 + 赛博朋克 + 复古未来主义
- 目标用户：喜欢叙事驱动、沉浸式体验的玩家
- 核心感觉：孤独、迷幻、压抑中的温暖

## 配色方案
### 主色板
- 背景：`bg-[#0a0a0f]` 极深蓝黑
- 表面：`bg-[#141420]` 暗海军蓝
- 卡片：`bg-[#1a1a2e]` 微亮海军蓝
- 主强调色：`text-[#00f0ff]` 霓虹青
- 次强调色：`text-[#ff0066]` 霓虹粉
- 第三强调：`text-[#7b2ff7]` 霓虹紫

### 中性色
- 正文：`text-[#e0e0e0]` 浅灰
- 次要文字：`text-[#8888aa]` 暗灰紫
- 禁用/暗淡：`text-[#444466]` 深灰紫

### 语义色（记忆情感）
- 悲伤：`#4488ff` 蓝
- 愤怒：`#ff3344` 红
- 快乐：`#ffcc00` 金
- 恐惧：`#44ff88` 绿
- 怀念：`#cc66ff` 紫
- 罪恶：`#880044` 暗红

### 语义色（UI）
- 成功：`#00ff88` 霓虹绿
- 警告：`#ffaa00` 琥珀
- 危险：`#ff3333` 红

## 字体规范
- 标题 H1：`text-2xl font-bold` 霓虹青色
- 标题 H2：`text-xl font-semibold` 浅灰色
- 正文：`text-sm` 浅灰色
- 数据/数值：`font-mono text-lg` 霓虹色
- 日志文字：`text-xs` 暗灰紫，打字机效果

## 间距系统
- 页面内边距：`p-4`
- 卡片内边距：`p-3`
- 组件间距：`gap-3`
- 紧凑间距：`gap-2`
- 底部安全区：`pb-16`（避开底部操作栏）

## 组件使用原则
- 通用 UI 组件优先使用 `@/components/ui/*`
- Button：所有操作按钮
- Card：信息卡片、记忆卡片、顾客信息
- Dialog：确认弹窗、选择分支
- Progress：鉴定进度、情感频谱
- Slider：频谱调频滑块
- Badge：状态标签、稀有度标签
- Tabs：页面内分段切换
- Textarea：日志区域
- Toast：操作反馈
- Separator：区域分割

## 导航结构
单页应用，游戏阶段通过状态驱动切换：
- Hub 视图：典当行主界面
- Dialog 视图：顾客对话
- Appraisal 视图：记忆鉴定小游戏
- Trading 视图：交易谈判
- Storage 视图：记忆存储柜

底部 TabBar：
- 典当行（Hub主界面）
- 存储柜（记忆库存管理）

## 小程序约束
- 禁止将大图片/视频打包到项目
- 所有动画使用 CSS animation（keyframes 在 .css 中定义）
- 记忆球效果用 CSS 渐变 + animation 实现
- 音效用 Taro.createInnerAudioContext 延迟加载
- 注意包体积，优先使用代码生成的视觉效果
